Flawless Cuts Barber Booking

Overview

This project is a polished single-page barber booking website for Flawless Cuts. It includes a branded hero section, service pricing, house-call pricing, a booking form, and a WhatsApp-based booking flow.

Features

- Responsive layout for desktop and mobile
- Interactive four-tile media gallery with modal preview
- Service and house-call pricing sections
- Booking form with date picker and validation
- WhatsApp booking handoff for quick customer contact
- Favicon and branded logo placement in the page

How to use

1. Open `index.html` in a browser to preview the site locally.
2. Update the contact details, pricing, and copy as needed.
3. Replace any image assets if you want different branding or new promotional artwork.

Project files

- `index.html` - Main page, layout, styles, and booking logic
- `assets/` - Image assets used by the page

Deployment

This site can be deployed as a static project on Vercel, Netlify, GitHub Pages, or Cloudflare Pages. For a simple deployment, upload the full project folder or connect the repository and publish the `index.html` file at the root.

Notes

- If you change the logo file name, update the references in `index.html` so the header, contact section, and tab icon continue to load correctly.
- The booking button currently sends messages to the business WhatsApp number configured in the page script.
